# Data
This folder contains all the data and configuration files for physical and 
simulated scanner

## Calibrations
Stores the generated calibration files for 

- camera (intrinsic, extrinsic, distortion, vignetting, white balance)
- projector (intrinsic, extrinsic, distortion, vignetting, response)


## Models
CAD model files for our calibration objects in .step format.


## Patterns
Different structured-light patterns that can be used for scanning:

- checker: simple checker board pattern for testing purposes
- color: rgb color patterns for color scanning
- gray: Gray code patterns
- mps: micro-phase shifting patterns
- ulp: unstructured light patterns
- white: plain white pattern for testing purposes

## Scenes
Scene templates for various render settings
